Diyet Takip Uygulaması
Bu Windows Forms uygulaması, kullanıcıların günlük beslenme alışkanlıklarını takip etmelerine, kalori alımlarını kontrol etmelerine ve beslenme hedeflerine ulaşmalarına yardımcı olmak için tasarlanmıştır.
